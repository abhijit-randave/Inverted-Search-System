# Inverted-Search-System
**Overview**
The Inverted Search System is a data structure-based project implemented in C that creates an index of words from multiple files and enables fast searching of keywords.

Features
File indexing
Keyword searching
Hash table implementation
Multiple file support
Efficient search operation
Technologies Used
C Programming
Hash Tables
Linked Lists
File Handling
Data Structures
Project Workflow
Read input files
Create word index
Store indexed data using hash table
Search keywords efficiently
Applications
Search engines
Text indexing
Document searching
Information retrieval systems
Advantages
Faster search operations
Efficient indexing mechanism
Supports multiple files
